# Домашнее задание 1
## Татьяна Авдеева
Поиск для данных с сайта [библиографии исторического танца]([url](https://bib.hda.org.ru)).

На данный момент реализован скорее для русского (задействована мультиязычная [модель](https://github.com/adbar/simplemma), которую легко адаптировать для других языков).

Я надеюсь, что я всё нужное выгрузила, но если вдруг что, изначально я работала с клонированным [открытым репозиторием](https://github.com/hda-technical/dancebooks) сайта библиографии.
## Как запускать

В файле [infopoisk_search](https://github.com/singing2owl/infopoisk2026/blob/main/infopoisk_search) реализован частотный и bm25-поиск с помощью словарей. Можно запустить программу и искать с помощью стандартного ввода.

В файле [infopoisk_search_matrix](https://github.com/singing2owl/infopoisk2026/blob/main/infopoisk_search_matrix) всё так же, только поиск реализован с помощью матриц.

В обоих файлах используется код из соседних файлов репозитория.
